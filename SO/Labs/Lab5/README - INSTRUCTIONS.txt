HOW
TO
RUN
----------------------------------------------------------------------
1. Run the makefile:
	$ make

2. Give permission to the test files to run as programs
(Right click >> "Properties" >> "Permissions" >> "Execute" checkbox):
	* Statistics mode only: test-statistics.sh
	* List mode only:       test-list.sh
	* Mixed mode:           test-all.sh

3. Run some test:
	$ ./test-statistics.sh
	$ ./test-list.sh
	$ ./test-all.sh
