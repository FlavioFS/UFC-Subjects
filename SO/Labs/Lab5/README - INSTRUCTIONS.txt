HOW
TO
RUN
----------------------------------------------------------------------
1. Run the makefile: $ make


2. Give permission to the files:
	-> "q2.sh"
	-> "q3-server.sh"
	-> "q3-client.sh"
	-> "q4-server.sh"
	-> "q4-client.sh"

to run as a programs (Right click >> "Properties" >> "Permissions"
>> "Execute" checkbox).


3. Run these scripts for automated test. E.g.: $ ./q2.sh

* Note: servers and clients must be in different console windows/tabs.
