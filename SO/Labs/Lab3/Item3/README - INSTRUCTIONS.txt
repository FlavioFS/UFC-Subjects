HOW
TO
RUN
-------------------------------------------------------------------
1. Run the makefile: $ make

2. Give permission to the files "run-server.sh" and "run-client.sh"
to run as programs (Right click >> "Properties" >> "Permissions" >>
"Execute" checkbox).

3. Open 2 separate console windows/tabs.

4. Run the shell script for the server: $ ./run-server.sh

5. Run the automated test for clients: $ ./run-clients.sh
